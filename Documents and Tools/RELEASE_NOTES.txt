Caelum Patched Release
======================

This release packages the canonical V3 refactor and adds a local MATLAB validation harness:

- validate_caelum_release.m

Recommended local validation:
1. Open the package folder in MATLAB.
2. Run:
   validate_caelum_release

This will:
- generate a synthetic V2 GPS/3D truth-aware log when available
- run caelum.analyzeLog(...)
- run caelum.runMonteCarloValidation(...)
- surface any remaining runtime integration errors in MATLAB directly

Notes:
- This package was hardened for handoff and packaging consistency.
- Runtime execution could not be performed in the current environment because MATLAB/Octave was unavailable here.
